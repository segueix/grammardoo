🔑 Positive → some · Negative & questions → any

👀 See them in action

🍎✅ I have some apples.
🍎❌ I don't have any apples.
🍎❓ Do you have any apples?
🥛✅ There is some milk.
🥛❌ There isn't any sugar.
STEP 2 OF 7 · LEARN

The Formula ✨

✅ ___ + some + noun
❌ ___ + not + any + noun
❓ ___ any + noun ?

What kind of nouns?

Use some/any with words you can't count exactly (lots of something) or with plurals:

Plural (count) apples, books, friends, cars
Uncountable water, milk, money, bread, sugar

🚫 Don't use some/any with a single thing → use a/an: "I have a book", not "I have some book".

🎁 Polite exception!

Use some in questions when you're offering or asking nicely:

☕Would you like some tea?
(offer)
💧Can I have some water, please?
(polite request)
